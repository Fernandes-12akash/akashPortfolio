About

I'am Akash

A passionate Full Stack Java Developer with hands-on experience in building scalable, enterprise-grade applications using Java, Spring Boot, Hibernate, PostgreSQL, Angular, Kafka, Elasticsearch, Redis, Firebase, Docker, and JWT. Skilled in both backend & frontend development, I have delivered projects ranging from Loyalty Management Systems and Member Portals to Online Assessment Platforms and cross-platform mobile applications, with expertise in microservices, role-based authentication, caching strategies, and real-time data processing. Always eager to learn, grow, and contribute to impactful software solutions.

Over the course of my career, I've worked across the full product lifecycle — from building the core Loyalty Management System and its companion Member Portal, to contributing to multiple POCs exploring new technologies and architectures, to stepping outside the backend-heavy world to build TrueHome, a cross-platform mobile app using React Native and NestJS. Each step has pushed me to pick up new tools and think more broadly about how systems connect, from event-driven microservices to mobile-first experiences.

Projects

Loyalty Management System (LMS – mSparsh)

Built an enterprise-grade loyalty platform with role-based user management, JWT security, notification engine, Kafka-driven event communication, Elasticsearch reporting, and Dockerized deployment. Delivered modules for customer onboarding, point allocation, redemption, and reporting.

Member Portal

A user-friendly Member Portal developed for the Loyalty Management System (LMS), enabling members to easily track earned loyalty points, redeemed rewards, transaction history, and account activities in real time. The portal provides a seamless experience for users to monitor loyalty benefits, explore available offers, redeem points, and stay engaged with the rewards program through a secure and responsive platform. Additionally, a configurable Product Store Module was introduced, allowing administrators to manage and customize reward products, categories, pricing, and redemption options dynamically based on business requirements.

Online Assessment Application

Developed a full-stack exam platform where admins can create, assign, and schedule tests. Implemented automated email notifications, login redirection to assigned tests, real-time scoring, and result dashboards. Results are auto-emailed to candidates, admins, and higher authorities.

TrueHome App

A cross-platform mobile application built using React Native for the frontend and NestJS for the backend, delivering a smooth, native-like experience across devices with a robust and scalable API layer.

Proof of Concepts (POCs)

Built and evaluated several POCs exploring new tools and patterns, including Redis for caching and session management and Firebase for authentication and real-time data sync, to assess their fit for production use cases.
